## Installation
Clone the Repo:
- git clone https://github.com/sonip98/taskmanagementLaravelVue.git
-  cd 
- composer install or composer update
- cp .env.example .env
- Set up .env file
- php artisan migrate
- php artisan serve
- Generate key
- npm install
- npm run dev
http://127.0.0.1:8000/
- register new user
- add new tasks
